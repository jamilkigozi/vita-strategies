# Database Schemas

Database schemas, migrations, and RBAC configurations for Vita Strategies platform.

## Structure

- **internal/** - Internal database schemas and migrations
- **client/** - Client database schemas and migrations  
- **tools/** - Database tools and utilities (Flyway, etc.)

## Features

- **Flyway Migrations** - Version-controlled database migrations
- **RBAC** - Role-based access control configurations
- **Seed Data** - Initial data for development and testing
- **Tools** - Database management scripts and utilities

## Getting Started

1. Configure your database connection in tools/flyway.conf
2. Review the migration files
3. Run migrations using the provided scripts

## Documentation

See docs/ directory for detailed schema design and migration guides.
